Summary: Lattitude vs X

The purpose of this project was to analyze how weather changes as you get closer to the quator. To accomplish this analysis we first pulled data from the OpenWeatherMap API to assumple a datase ton over 500 cities.A fter assembling the dataset, we used Matplotlib to plot various aspects of the weather vs latitude. Factors we looked at includedt:temperature, cloudiness, wind speed, and humidity. Thsis site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any correlations witnessed.
